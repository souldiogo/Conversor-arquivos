Esse é apenas uma das curiosidade sobre desenvolvimento web. Você sabia que a primeira página web foi criada por Tim Berners-Lee em 1991? Ela era um simples documento HTML que explicava o projeto da World Wide Web. Desde então, a web evoluiu significativamente, passando de páginas estáticas para aplicações dinâmicas e interativas.
